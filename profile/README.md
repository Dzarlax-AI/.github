# Dzarlax AI

Personal AI-powered ecosystem built around everyday automation, health tracking, and smart home control. Everything runs on a self-hosted Hetzner VPS + Synology NAS stack.

## Projects

### AI Assistants
- **[Personal Assistant](https://github.com/dzarlax/personal_assistant)** — Telegram bot with multi-LLM support (Claude, Gemini, DeepSeek), tool use, calendar and health briefings
- **[Personal Memory](https://github.com/Dzarlax-AI/personal_memory)** — Semantic long-term memory server with Qdrant vector search, powering context across all AI interactions

### Smart Home
- **[HomeDash](https://github.com/dzarlax/homedash)** — 7" ESP32-S3 touchscreen dashboard: weather, calendar, health, tasks, news, transport, and Home Assistant controls
- **[ESP32 Bridge](https://github.com/dzarlax/esp32-bridge)** — Go microservice aggregating all data sources into a single API for the display, with OTA firmware delivery

### Health & Wellness
- **[Health Dashboard](https://github.com/dzarlax/health_dashboard)** — Apple Health data ingestion, sleep/HR/HRV/SpO2 tracking, readiness scoring, web UI and MCP server

### Information
- **[Evening News](https://github.com/dzarlax/rss-summarizer)** — AI news aggregator: RSS feeds, web scraping, LLM summarization and categorization
- **[City Dashboard](https://github.com/dzarlax/city-dashboard)** — Serbian public transit: schedules, routes and real-time arrivals for Belgrade, Novi Sad and Niš

### Productivity
- **[Todoist Bot](https://github.com/dzarlax/todoist_bot)** — Telegram-to-Todoist task capture with message concatenation
- **[Streamdeck AI Plugin](https://github.com/Dzarlax-AI/Streamdeck-AI-plugin)** — Stream Deck integration for AI assistants

## Stack

`Go` `Python` `FastAPI` `C/C++` `ESP-IDF` `LVGL` `PostgreSQL` `Qdrant` `Docker` `Traefik` `GitHub Actions`
