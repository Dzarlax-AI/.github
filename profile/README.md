# Dzarlax AI

Personal AI-powered ecosystem — assistants, memory, health tracking and intelligent information processing. Self-hosted on a VPS with NAS backup.

## Projects

### AI Assistants
- **[Personal Assistant](https://github.com/Dzarlax-AI/personal_assistant)** — Telegram bot with multi-LLM support (Claude, Gemini, DeepSeek), tool use, calendar and health briefings
- **[Personal Memory](https://github.com/Dzarlax-AI/personal_memory)** — Semantic long-term memory server with Qdrant vector search, powering context across all AI interactions

### AI-Powered Data
- **[Health Dashboard](https://github.com/Dzarlax-AI/health_dashboard)** — Apple Health data ingestion, sleep/HR/HRV/SpO2 tracking, readiness scoring, web UI and MCP server for AI assistants
- **[Evening News](https://github.com/Dzarlax-AI/evening-news)** — News aggregator: RSS feeds, web scraping, LLM summarization and categorization

### Integrations
- **[Streamdeck AI Plugin](https://github.com/Dzarlax-AI/Streamdeck-AI-plugin)** — Stream Deck integration for AI assistants

## Stack

`Go` `Python` `FastAPI` `PostgreSQL` `Qdrant` `Docker` `Traefik`
